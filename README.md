1. 얼굴 인식 후 빨간 박스 생성

2. 얼굴 인식 박스별 고유ID 값을 박스 센터에 할당

3. 고유ID 값이 중앙 빨간선(Counting Line)을 오른쪽에서 왼쪽으로 이동하면 화면 좌측 하단에 인원수가 +1 증가
