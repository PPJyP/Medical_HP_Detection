  * Stacking이란? 
  
  :앙상블 기법은 머신러닝 모델의 성능을 향상할 때 많이 쓰이는 기법으로, 각 모델에서 예측한 값들을 가지고 최종 모델을 이용하여 한 번 더 target을 예측하도록 하여 모델의 예측 성능을 향상시킬 수 있는 기법이다.
